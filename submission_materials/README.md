# Submission Materials

This directory contains editable files for an Elsevier-style submission package.

- `highlights.md`: five short highlights.
- `cover_letter.md`: conservative cover letter for *Solar Energy*.
- `graphical_abstract.png`: vertical graphical abstract generated from the editable template in `vertical_graphical_abstract_template/`.
- `graphical_abstract.pdf`: PDF export of the same vertical graphical abstract.
- `graphical_abstract_note.md`: guidance, build route, and claim boundary for the graphical abstract.
- `vertical_graphical_abstract_template/`: source template, configuration, and builder script for regenerating the graphical abstract.
- `data_availability_statement.md`: draft data/code availability statement.
- `declaration_of_interests.md`: declaration text.
- `funding_statement.md`: funding statement from the submission metadata.
- `author_contributions_credit.md`: draft CRediT roles to confirm with coauthors.
- `reviewer_risk_response_memo.md`: internal memo for revision, not a direct upload file.

The previous horizontal graphical abstract is archived under
`archive/graphical_abstract_horizontal_replaced_20260611/`.
